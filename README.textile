h1. clj-base64

Functions to encode and decode base64 strings.

h2. Usage

<pre>
(use 'remvee.base64)

(encode-str "I 8> clojure!") ; => "SSA4PiBjbG9qdXJlIQ=="
(decode-str "SSA4PiBjbG9qdXJlIQ==") ; => "I 8> clojure!"
</pre>
